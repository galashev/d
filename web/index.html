Разработать серверное приложение, осуществляющие маршрутизацию http запросов между сервисами из рееста.
Приложение должно выполнять следующие задачи:
1. добавлять или удалять сервисы без перезагрузки
2. принимать и по uri проксировать http запросы в соответствующие сервисы (если сервис не найден, возвращается ошибка 404)
3. на get запроса /admin/services возвращать список сервисов
4. на get запроса /admin/shutdown выключать сервер


сервисом является любой класс, реализующий интерфейс
interface IService {
/*уникальное имя сервиса*/
String getName();
/*обработка запроса*/
void handleRequest(HttpServletRequest request, HttpServletResponse response);
}

добавление и удаление сервисов осуществляется путем копирование и удаление *.class файлов
http запросы с сервисом прокисруются по следующему правилу: /имя класса, реализующего IService /параметры
параметры могут передаваться как методом GET, так и методом POST
